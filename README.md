## What To Customize

- Replace `connection_schema()` with the datasource-specific connection fields.
- Replace `request_schema()` with the per-request inputs your datasource accepts.
- Replace `output_schema()` with the fields returned by `run()`.
- Implement the real datasource logic inside `run(request)`.
- Keep reading connection values from `self._connection` first, then fall back to `self.params`.

## V2 Rules

- Do not put `connection_name` in `connection_schema()`.
- Do not add `get_connection_data()`, `get_icon()`, or `fetch()` unless you are intentionally keeping legacy v1 compatibility.
- Keep `icon()` as a classmethod that reads `icon.svg`.
